VS Audio Editor

How do you use VS Audio Editor?

First, select the effect you want to apply by clicking one of the buttons:

  • Blue (Pitch): Change the pitch of the audio
  • Red (Volume): Adjust the volume
  • Yellow (Reverse): Reverse the audio playback

After selecting an effect, upload the MP3 file you'd like to edit. Once the file is uploaded, click the same button again to apply the effect.
